Trace une carte isobare

Instructions

  1. Survolez un point pour voir la valeur de pression (kpa) mesurée à cette station météo.
  2. Cliquez sur un point, puis sur un autre de même valeur pour tracer une ligne.
  3. Placez le marqueur A dans la zone de haute pression (anticyclone) et le marqueur D dans la zone de basse pression (dépression).
  4. Cliquez sur 'Valider' pour vérifier votre carte.
